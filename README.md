# Moha E-Commerce App 🛒

### Web app project is developed to simulate the functionalities of real-world ecommerce web apps. ⚡🛒

## Technologies ⚙️
- React.js (Javascript) 
- Express.js (Typescript)
- MongoDB 🌱
- Prisma (ORM)

## Features 🪶
- Friendly user interface 👤
- Authentication & Authorization 🔐
- Handling orders 🛒
- favorite products ❤️
- a lot more.. ✨
